# Preuves techniques

Ce dossier centralise toutes les preuves d'exécution effective des actions documentées.

## Types de preuves à archiver

| Type | Format attendu | Nommage suggéré |
|---|---|---|
| Captures d'écran | PNG ou JPG | `SXX_capture_NN_description.png` |
| Exports de configuration | TXT, JSON, XML | `SXX_export_NN_description.ext` |
| Journaux applicatifs | TXT | `SXX_log_NN_description.txt` |
| Fichiers de configuration source | Selon outil | `SXX_config_NN_description.ext` |
| Schémas graphiques | PNG plus fichier source draw.io | `SXX_schema_NN_description.png` |
| Vidéos de démonstration | MP4 | `SXX_video_NN_description.mp4` |

`SXX` désigne le numéro de séance (S01, S02, etc.). `NN` est un compteur séquentiel par séance.

## Règle d'or

Une action documentée sans preuve archivée dans ce dossier ou explicitement référencée vers une preuve externe est considérée comme non réalisée par la consigne de Vanessa Robbe, chargée du cours.
