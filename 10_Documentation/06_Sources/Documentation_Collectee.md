# Documentation collectée

Toutes les ressources externes consultées pendant le projet sont consignées ici avec leur justification d'utilité.

## Sources principales

| Source | Type | Pourquoi utile |
|---|---|---|
| docs.fogproject.org | Documentation officielle | Référence officielle pour l'installation, la configuration et l'usage de FOG. Premier point d'entrée pour toute question technique. |
| forums.fogproject.org | Forum communautaire | Cas pratiques, résolution de problèmes courants, échanges entre administrateurs. |
| github.com/FOGProject/fogproject | Code source et issues | Suivi des versions, lecture des notes de version, vérification de la maintenance active du projet. |
| microsoft.com (centre d'évaluation) | ISO d'évaluation | Téléchargement de l'ISO Windows 10 Pro 22H2 en version d'évaluation 90 jours, conforme à l'usage pédagogique. |
| virtualbox.org/manual | Documentation officielle VirtualBox | Configuration des modes réseau, en particulier le mode Bridged et le réseau Host-only. |

## Règle de consignation

Toute source consultée pendant le projet et ayant influencé une décision technique ou une procédure est ajoutée à ce tableau. Une source citée dans le rapport final sans être référencée ici est considérée comme non sourcée.
